# viewport


## Project setup
```
npm i h-vw -S  
or
yarn add h-vw
```
## Project usage
```
import 'h-vw'
```

## result
document.documentElement.clientWidth === 375

## other usage
```
import vw from 'h-vw'
vw(480);
```
## result
document.documentElement.clientWidth === 480
